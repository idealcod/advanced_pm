# Advanced PM System

A backend service for managing events using Go, PostgreSQL, and Redis, designed with clean architecture principles.

---

##  Problem

Managing event-based data efficiently requires:
- fast access to data
- reliable storage
- scalable architecture

This project demonstrates how to build a backend system that solves these problems using modern technologies and best practices.

---

##  Features

- Event management (create, read, update)
- PostgreSQL database integration
- Redis caching for performance
- Middleware support
- Clean and modular Go architecture
- Scalable project structure

---

##  Technology Stack

- Go (Golang)
- PostgreSQL
- Redis
- Standard Go libraries
